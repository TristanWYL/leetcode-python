# Solution

```python
class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        index_slot = 0
        for num in nums:
            if num != val:
                nums[index_slot] = num
                index_slot += 1
        return index_slot
```

## complexity

- time: $O(n)$
- space: $O(1)$
